# Parche Plan U

Parche Plan U es una aplicación web pensada para ayudar a organizar la vida universitaria de forma más sencilla y colaborativa. La idea es que los estudiantes puedan crear parches y planes dentro de un mismo espacio, y que sea facil consultarlos y agendarse pa integrarse.

Muchas veces los planes entre estudiantes se comparten en varios lugares al mismo tiempo, como whatsapp, y en recesos. Al final con la información dispersa suelen no concretarse las salidas. Parche Plan U busca centralizar todo en una sola plataforma donde los usuarios puedan ver rápidamente qué planes hay disponibles, revisar los detalles de cada uno y decidir si les interesa participar.

Los usuarios pueden registrarse e iniciar sesión, lo que les permite crear y gestionar sus propias publicaciones. A través de formularios sencillos pueden publicar nuevos parches o planes, agregando detalles relevantes para que los demás estudiantes entiendan de qué se trata la actividad.

La plataforma también permite visualizar páginas de detalle donde se muestra toda la información de cada propuesta. Además, incluye secciones de rankings para destacar las actividades más populares.

El sistema utiliza persistencia local de datos para mantener cierta información guardada en el navegador.

En general, Parche Plan U funciona como una herramienta para facilitar la coordinación de actividades entre estudiantes, implementa diferentes conceptos del desarrollo de ingenieria web, como el manejo de formularios, la organización de vistas, la gestión de datos en el cliente y el diseño de interfaces enfocadas en la experiencia del usuario.
