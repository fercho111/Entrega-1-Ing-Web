# Parche Plan U

Parche Plan U es una aplicación web pensada para organizar la vida universitaria de forma colaborativa, permitiendo que los estudiantes publiquen y consulten *parches* (actividades puntuales) y *planes* (propuestas más estructuradas) dentro de un mismo espacio. La plataforma centraliza la información para que cada usuario pueda descubrir opciones según su interés, revisar detalles de cada propuesta y tener una vista más clara de lo que está pasando en la comunidad.

En términos funcionales, el proyecto cubre el flujo completo de uso: registro e inicio de sesión, creación de nuevos parches y planes mediante formularios, visualización de listados y páginas de detalle, y consulta de rankings para resaltar la participación o relevancia de las publicaciones. Además, gestiona estados vacíos y persistencia local de datos para mantener una experiencia fluida, enfocada en facilitar la coordinación y la toma de decisiones al momento de armar o unirse a actividades.
